How to use it

Open the project in ConsoleApplication11/ConsoleApplication11/ConsoleApplication11.vcxproj

build the visual c++ project

Run the visual c++ project

It will ask to record the sound giving some silent sound at the start and end of the recording of 4 seconds

After it shows it has stopped recording on the terminal, press enter to view the result

NOTE:- everytime the input is taken from the user it is stored in out.wav and out.txt for sound and its samples respectively
ALSO NOTE:- all the ai's ,ci's and ri's are stored in the sample folder where the cpp file is with their name along with vowel and iteration number
