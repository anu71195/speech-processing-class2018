How to use it

Open the project in ConsoleApplication7/ConsoleApplication7/ConsoleApplication7.vcxproj

build the visual c++ project

Run the visual c++ project

It will ask to record the sound giving some silent sound at the start and end of the recording of 4 seconds

After it shows it has stopped recording on the terminal, press enter to view the result

NOTE:- It will only tell if it 1 or 6. So it is also expected that the user gives the input of only 1 or 6.
ALSO NOTE:- everytime the input is taken from the user it is stored in out.wav and out.txt for sound and its samples respectively
